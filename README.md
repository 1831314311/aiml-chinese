# 中文 aiml 机器人实现

使用结巴分词和修改过的 python aiml 包实现

## 使用

1. 需要 python 2 环境

2. 使用 pip 安装依赖

   ```shell
   pip install jieba
   ```

   执行主程序进行对话

   ```shell
   python main.py
   ```

   ​

3. 如果需要部署网页服务器，还需安装依赖

   ```shell
   pip install web.py
   ```

   然后运行 application 程序

   ```shell
   python run application.py
   ```

   访问网页 `localhost:8080`
